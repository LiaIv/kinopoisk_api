# Кинопоиск App

Веб-приложение для работы с базой данных фильмов Кинопоиска. Позволяет просматривать популярные фильмы, искать по названию, добавлять в избранное и просматривать детальную информацию о каждом фильме.

## Функционал

- **Список фильмов** — популярные фильмы из API Кинопоиска
- **Поиск** — поиск фильмов по названию через API
- **Избранное** — сохранение фильмов в избранное (LocalStorage)
- **Детали фильма** — описание, рейтинг, ссылка на Кинопоиск

## Технологии

- React
- React Router
- Redux Toolkit
- Material UI
- Storybook
- Vite
- ESLint + Prettier

## Установка и запуск

```bash
npm install
npm run dev
```

## Скрипты

- `npm run dev` — запуск dev-сервера
- `npm run build` — сборка проекта
- `npm run lint` — проверка кода ESLint
- `npm run format` — форматирование Prettier
- `npm run storybook` — запуск Storybook
- `npm run deploy` — деплой на GitHub Pages
